# QuestMax ⚔️

QuestMax is a mobile-first web application that turns your boring summer into an epic adventure. It generates real-life "side quests" tailored to your preferences, budget, and party size. 

Designed with a classic 2000s medieval MMO aesthetic, QuestMax makes stepping out of your comfort zone feel like leveling up.

## Features (MVP)
* **Medieval MMO UI**: Experience a nostalgic pixel-art interface with stone buttons, parchment scrolls, and torchlight glows.
* **Quest Board**: Browse a full list of summer side quests, categorized by rarity (Chill, Challenging, Epic, Legendary).
* **Quest Wizard**: Answer a short questionnaire about your vibe, budget, and party size to receive 5 personalized quest recommendations.
* **Loot Reveal**: Enjoy a satisfying retro chest-opening animation when your quests are generated.
* **Retro Audio**: Authentic 8-bit sound effects for every tap, swipe, and reveal (with a mute toggle).

## Tech Stack
* HTML5
* CSS3
* Vanilla JavaScript

## Setup Instructions
1. Clone the repository.
2. Open `index.html` in your browser. No build steps required for the MVP!
3. For the best experience, use Chrome DevTools and emulate a mobile device (e.g., iPhone 12/13 or Pixel 5).
